### Step 7: Complete the Hackathon Form

**What did you build?**
```
A production-ready Custom Agent for GitLab Duo Agent Platform that:
1. Automatically scans MRs for security vulnerabilities
2. Uses Claude AI to analyze code and detect issues
3. Generates intelligent, safe fixes
4. Posts comprehensive security reviews to MRs
5. Creates auto-fix commits on MR branches
```

**How does it work?**
```
When a merge request is created, Security Guardian is automatically triggered
via GitLab webhook. It uses Claude 3.5 Sonnet to analyze the code diff and
detect vulnerabilities across 8+ categories (hardcoded secrets, SQL injection,
insecure patterns, etc.). The agent then generates safe fixes and posts a
detailed security review comment to the MR with findings, severity levels,
and actionable recommendations.
```

**Key Technologies:**
```
- GitLab Duo Agent Platform (official)
- Claude 3.5 Sonnet (AI backbone)
- Model Context Protocol (MCP)
- JavaScript/Node.js
```

**Why you built it?**
```
To accelerate security reviews and reduce vulnerability remediation time by
deploying an intelligent AI agent that understands security context, generates
safe fixes, and educates developers about security risks.
```
